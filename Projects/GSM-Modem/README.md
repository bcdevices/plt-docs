# Test Plans for AT-based GSM modems

Example PLT test plans for GSM modems with AT-based interface.

- `atmodem-gsm.yaml` : GSM Modem identification

## PLT-200A connection

| PLT-200A      | Modem       |
|---------------|:-----------:|
| UART0\_RXD    | RXD         |
| UART0\_TXD    | TXD         |
| UART0\_CTS    | CTS         |
| UART0\_RTS    | RTS         |
| GND           | GND         |
